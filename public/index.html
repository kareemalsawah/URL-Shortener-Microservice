This Microservice allows you to

Input a url and get a shortened url that will redirect you to the original page

Example

https://nickel-trombone.glitch.me/api/?website=https://www.google.com

will return

{

link: https://www.google.com,

short: https://nickel-trombone.glitch.me/api/?website=https://hi.com/8365

}

now if you go to

https://nickel-trombone.glitch.me/api/?website=https://hi.com/8365

You will be redirected to

https://www.google.com
